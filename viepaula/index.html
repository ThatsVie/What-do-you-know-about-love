About Me

Hi, I’m Vie (/vi/). I’m passionate about building web experiences that are accessible, inclusive, and impactful. With a background as a licensed massage therapist, I bring a unique attention to detail and care to every project, focusing on creating user-centered designs that foster connection. I also leverage AI tools like ChatGPT to brainstorm ideas, streamline workflows, and create detailed README.md files with a focus on usage and testing. My goal is to create technology that not only functions smoothly but also leaves a positive, lasting impact.

My Portfolio

Pug Smile School

An interactive web app featuring pugs, created to spread positivity and joy. Focused on accessibility and responsiveness, this lighthearted project is perfect for dog lovers. The app includes three web pages built with Bootstrap, showcasing skills in HTML, CSS, and responsive design. Based on Figma design files, each page was developed with creative freedom to incorporate pug-themed elements, resulting in a visually engaging and accessible experience for all users.

Realms of Reality Portal

A custom Chrome Extension designed to enhance productivity and personalization every time users open their browsers. Featuring dynamic, high-resolution space-themed backgrounds, a real-time weather widget, and a rotating display of inspirational quotes, this portal offers a visually engaging and uplifting experience. Additional functionality includes a prominent time display, a fully functional Google search bar, and a hidden menu with quick access to useful links such as Tarot, Runes, I Ching, and horoscopes. By blending creativity with practicality, the Realms of Reality Portal challenges users to reflect on their perception of reality while providing the tools they need to navigate their day.

AirBnB Clone

A collaborative project that recreated Airbnb's core functionalities over four iterations, progressively enhancing the platform's features and user experience. Built using Python, Flask, MySQL, HTML, CSS, and JavaScript, the project introduced dynamic content, search filters, and API integrations. It involved implementing both front-end and back-end functionalities, focusing on optimizing performance, ensuring system reliability, and improving data flow through RESTful APIs.
